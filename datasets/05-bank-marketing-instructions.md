# Bank Marketing Dataset Instructions

## Dataset Overview

This dataset contains marketing campaign data from a Portuguese banking institution, including customer demographics, campaign details, and subscription outcomes. Perfect for analyzing marketing effectiveness and customer behavior patterns.

## Dataset Files

- **bank-additional-full.xlsx**: Complete dataset with all customer and campaign information

## Variables Description

The dataset includes:
- **Customer Demographics**: Age, job, marital status, education
- **Financial**: Balance, housing loan, personal loan
- **Campaign**: Contact type, month, day of week, duration
- **Previous Campaigns**: Previous contacts, outcome
- **Outcome**: Subscription (yes/no)

## Use Cases

This dataset is suitable for:
- Analyzing campaign effectiveness
- Segmenting customers by characteristics
- Identifying factors that influence subscription
- Calculating conversion rates and metrics
- Creating marketing analytics dashboards

## Excel Analysis Tips

- Use PivotTables to analyze conversion rates by segment
- Create charts comparing subscription rates across groups
- Calculate key marketing metrics (conversion rate, segment performance)
- Use conditional formulas for segmentation analysis
- Build visualizations highlighting customer segments
- Create dashboards for campaign performance
