# Stock Price Dataset Instructions

## Dataset Overview

This dataset contains daily stock prices for major technology companies (Amazon, Apple, Facebook, Google, Netflix) including open, high, low, close prices and trading volume. Ideal for practicing time series analysis and trend identification.

## Dataset Files

- **Amazon.xlsx**: Amazon stock price data
- **Apple.xlsx**: Apple stock price data
- **Facebook.xlsx**: Facebook stock price data
- **Google.xlsx**: Google (Alphabet) stock price data
- **Netflix.xlsx**: Netflix stock price data

## Variables Description

Each stock file includes:
- **Date**: Trading date
- **Open**: Opening price
- **High**: Highest price of the day
- **Low**: Lowest price of the day
- **Close**: Closing price
- **Volume**: Trading volume
- **Adj Close**: Adjusted closing price

## Use Cases

This dataset is suitable for:
- Time series analysis
- Analyzing price trends and patterns
- Comparing performance across stocks
- Calculating returns and volatility
- Identifying seasonal patterns
- Creating time series visualizations

## Excel Analysis Tips

- Create time series charts (line charts)
- Calculate daily returns: (Close - Previous Close) / Previous Close
- Calculate moving averages for trend identification
- Compare performance across different stocks
- Use date functions for time-based analysis
- Create dashboards with multiple time series
- Calculate volatility and risk metrics
