# Superstore Sales Dataset Instructions

## Dataset Overview

This dataset contains sales data for a fictional retail company (Superstore), including product information, orders, customers, and financial metrics. The data is ideal for practicing Excel fundamentals with real business context.

## Dataset Files

- **SuperStoreUS-2015.xlsx**: Main dataset file containing sales transactions

## Variables Description

The dataset includes information about:
- **Orders**: Order ID, order date, ship date, shipping mode
- **Customer**: Customer name, segment, geographic location
- **Product**: Product name, category, subcategory
- **Sales Metrics**: Sales amount, quantity, discount, profit
- **Geography**: Country, state/region, city, postal code

## Use Cases

This dataset is suitable for:
- Calculating total sales and profits
- Analyzing sales trends over time
- Identifying top-performing products and regions
- Creating summary reports and dashboards
- Practicing basic Excel formulas and functions

## Excel Analysis Tips

- Use PivotTables to summarize sales by category, region, or time period
- Create charts to visualize trends
- Use SUMIFS, COUNTIFS for conditional calculations
- Practice date functions for time-based analysis
- Build interactive dashboards with slicers
