# Adult Census Income Dataset Instructions

## Dataset Overview

This dataset contains demographic, social, and economic attributes from the 1994 U.S. Census database. It is rich in categorical and numerical variables, ideal for exploring relationships and patterns across demographic groups.

## Dataset Files

- **adult.xlsx**: Main dataset file containing census records

## Variables Description

The dataset includes:
- **Demographics**: Age, sex, race, native country
- **Education**: Education level, years of education
- **Employment**: Work class, occupation, hours per week
- **Income**: Income level (binary classification: >50K or <=50K)
- **Marital Status**: Marital status, relationship status
- **Capital**: Capital gain, capital loss

## Use Cases

This dataset is suitable for:
- Exploratory data analysis (EDA)
- Identifying relationships between demographics and income
- Segmenting population by various characteristics
- Creating summary statistics and visualizations
- Understanding income distribution patterns

## Excel Analysis Tips

- Use PivotTables to explore income distribution by demographics
- Create charts comparing income levels across groups
- Use conditional formatting to highlight patterns
- Practice data segmentation and grouping
- Build summary tables for different demographic segments
