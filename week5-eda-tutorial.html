1) Setup and Data Cleaning (Clear Step-by-Step for Mac + Google Sheets)

Use this section first. If you are on a MacBook and using Google Sheets, follow Path B.

Path A (Excel on Mac)

  1. Open Medical-EDA.xlsx in Excel.
  2. Click any cell in the data range, then press Cmd + T to create a Table.
  3. In the Table Design tab, rename table to MedicalEDA.
  4. Data cleaning:
    • Trim text in Region, TreatmentGroup, DiagnosisGroup.
    • Normalize labels: a/b to A/B; uppercase diagnosis names to proper case.
    • Set data types: LDL and AdherenceScore as Number.
    • Remove blank rows and exact duplicates.

Path B (Google Sheets on MacBook)

  1. Open Google Sheets -> File -> Import -> Upload Medical-EDA.csv (recommended) or Medical-EDA.xlsx.
  2. After import, confirm row 1 is headers, then turn on filters: Data -> Create a filter.
  3. Add helper columns to clean text (example assumes Region is in column E):
    =PROPER(TRIM(E2))
    Do the same pattern for TreatmentGroup and DiagnosisGroup:
    • Treatment normalize example: =UPPER(TRIM(H2))
    • Diagnosis normalize example: =PROPER(TRIM(G2))
  4. Convert numeric-text fields to numbers:
    • LDL: =VALUE(TRIM(K2))
    • AdherenceScore: =VALUE(TRIM(N2))
  5. Copy helper-column formulas down, then copy -> paste special -> Values only back to original columns.
  6. Remove duplicates: Data -> Data cleanup -> Remove duplicates.
  7. Remove blank rows: filter by PatientID blank, delete visible blank rows, then clear filter.

Quick quality checks (both tools)

  • TreatmentGroup should only contain A or B.
  • Region should only contain North, South, East, West (plus optional missing if kept).
  • LDL and AdherenceScore must be numeric (right-aligned in Sheets by default).

Cleaned example used below: 233 records (after filtering segmentation to valid regions only).

2) Descriptive Statistics (Detailed Step-by-Step)

Goal: Build a descriptive-statistics table for BaselineSBP, FollowupSBP, LDL, and BMI.

Step A - Build the summary table layout

  1. Create a new sheet named Descriptive_Stats.
  2. In row 1, enter:
    • A1 = Metric
    • B1 = BaselineSBP
    • C1 = FollowupSBP
    • D1 = LDL
    • E1 = BMI
  3. In column A, enter:
    • A2 = Mean
    • A3 = Median
    • A4 = Std Dev
    • A5 = Min
    • A6 = Max
    • A7 = N

3) Summary Tables (Detailed Step-by-Step)

Step A - Mean outcomes by TreatmentGroup

  1. Create a sheet named Summary_Tables.
  2. Create this small table:
    • A1 = TreatmentGroup, B1 = Mean FollowupSBP, C1 = Mean LengthOfStay
    • A2 = A, A3 = B
  3. Fill formulas for row A and copy down to row B.

Google Sheets (MacBook) formulas

B2: =AVERAGEIFS('Medical-EDA'!J2:J,'Medical-EDA'!H2:H,$A2)
C2: =AVERAGEIFS('Medical-EDA'!O2:O,'Medical-EDA'!H2:H,$A2)

Assumption: H=TreatmentGroup, J=FollowupSBP, O=LengthOfStay.

Excel Table formulas

B2: =AVERAGEIFS(MedicalEDA[FollowupSBP], MedicalEDA[TreatmentGroup], $A2)
C2: =AVERAGEIFS(MedicalEDA[LengthOfStay], MedicalEDA[TreatmentGroup], $A2)

Expected values:

TreatmentGroupMean FollowupSBPMean LengthOfStay
A124.924.96
B135.996.05

Slide message: Group A has lower follow-up SBP and shorter average stay than Group B.

Step B - Region x Diagnosis count matrix

  1. Create matrix headers:
    • Rows (A6:A9): East, North, South, West
    • Columns (B5:D5): Diabetes, Hypertension, Respiratory
  2. In B6, use one formula and copy right/down for the full matrix.

Google Sheets formula

B6: =COUNTIFS('Medical-EDA'!E2:E,$A6,'Medical-EDA'!G2:G,B$5)

Assumption: E=Region, G=DiagnosisGroup.

Excel Table formula

B6: =COUNTIFS(MedicalEDA[Region], $A6, MedicalEDA[DiagnosisGroup], B$5)

Copy from B6 across to D6, then down to row 9.

8) Inferential Add-on (CI & p-values)

Goal: Add one inferential block to support your descriptive findings.

8.1 95% CI for mean FollowupSBP (overall)

  1. Create a sheet Inferential.
  2. Set labels in column A: Mean, SD, N, t*, Margin Error, CI Lower, CI Upper.
  3. Compute values for FollowupSBP using formulas below.

Google Sheets (MacBook)

B2 (Mean): =AVERAGE('Medical-EDA'!J2:J)
B3 (SD):   =STDEV.S('Medical-EDA'!J2:J)
B4 (N):    =COUNT('Medical-EDA'!J2:J)
B5 (t*):   =T.INV.2T(0.05,B4-1)
B6 (ME):   =B5*B3/SQRT(B4)
B7 (CI L): =B2-B6
B8 (CI U): =B2+B6

Assumption: J = FollowupSBP.

Excel Table version

B2: =AVERAGE(MedicalEDA[FollowupSBP])
B3: =STDEV.S(MedicalEDA[FollowupSBP])
B4: =COUNT(MedicalEDA[FollowupSBP])
B5: =T.INV.2T(0.05,B4-1)
B6: =B5*B3/SQRT(B4)
B7: =B2-B6
B8: =B2+B6

8.2 95% CI by Treatment Group (A vs B)

  1. Create two filtered ranges for FollowupSBP: one for A and one for B.
  2. Repeat 8.1 formulas separately for each group.

Slide use: compare the two CI centers and interval overlap.

8.3 p-value for difference in means (A vs B)

  1. Place Treatment A FollowupSBP values in one column and Treatment B in another.
  2. Use a two-sample t-test (two-tailed, unequal variance).

Formula

=T.TEST(rangeA, rangeB, 2, 3)
  • 2 = two-tailed test
  • 3 = unequal variances (Welch test)

8.4 Optional categorical p-value (Readmission vs Treatment)

  1. Create a 2x2 observed table: TreatmentGroup x Readmission30d.
  2. Compute expected counts (or use analysis add-on/workflow).
  3. Run chi-square test:
=CHISQ.TEST(observed_range, expected_range)

8.5 Decision and interpretation template (for slides)

  • If p < 0.05: “There is statistical evidence of a difference between groups.”
  • If p >= 0.05: “No statistically significant difference detected at alpha = 0.05.”
  • Always pair p-value with effect size direction and CI interpretation.

Ready-to-use sentence: “Treatment A showed lower mean FollowupSBP than Treatment B; the estimated mean difference was statistically significant (p < 0.05), with 95% CI supporting lower follow-up pressure in Group A.”
